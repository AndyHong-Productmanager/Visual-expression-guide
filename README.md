# 🎬 AI Visual Prompt Master Guide

**AI 이미지/영상 생성을 위한 시각 용어 종합 가이드 — 150+ 용어, SVG 일러스트 포함**

> 나노바나나, Flow, Midjourney, Runway, Kling, Sora 등에서 프롬프트를 쓸 때  
> "예쁘게 해줘" 대신 **정확한 용어 하나**가 결과물의 퀄리티를 결정한다.

---

## 이게 뭔가

바이브코딩으로 앱을 만들 듯, AI 이미지/영상도 "바이브 프롬프팅"으로 만드는 시대다.  
그런데 코딩에 변수명이 중요하듯, 프롬프트에도 **정확한 용어**가 중요하다.

이 가이드는 영화/사진 현장에서 실제로 쓰는 카메라 워크, 필름, 조명, 연출 용어를 AI 프롬프트 관점에서 재정리한 **단일 HTML 파일**이다. 인터넷 없이 브라우저만 있으면 바로 볼 수 있다.

각 용어마다 **한국어 설명 + 영문 프롬프트 예시 + SVG 일러스트**를 포함한다.

## 포함된 내용 (12개 섹션, 150+ 용어)

### 📷 카메라 워크 (섹션 1~5)

| 섹션 | 내용 | 주요 용어 |
|------|------|-----------|
| 샷 사이즈 | 프레임 안 피사체 비율 | EWS, Wide, Medium, CU, ECU, Insert |
| 카메라 앵글 | 수직적 촬영 각도 | Eye Level, Low/High Angle, Bird's Eye, Dutch, OTS |
| 무브먼트 | 카메라 움직임 | Pan, Tilt, Dolly, Zoom, Orbit, Tracking, Crane, Handheld |
| 렌즈 & 초점 | 렌즈 특성과 초점 | Shallow DOF, Anamorphic, Macro, Wide Angle, Rack Focus |
| 구도 | 프레임 내 배치 | Rule of Thirds, Symmetry, Leading Lines, Frame in Frame |

### 🎞️ 비주얼 스타일 (섹션 6~8)

| 섹션 | 내용 | 주요 용어 |
|------|------|-----------|
| 필름 시뮬레이션 | 필름 색감 재현 | Nostalgic Neg, Classic Neg, Portra 400, Ektar, CineStill 800T, Velvia, Gold, Tri-X |
| 색보정 | 컬러 그레이딩 | Teal & Orange, Desaturated, High/Low Key, Bleach Bypass, Cross Process |
| 조명 | 빛의 방향과 질감 | Golden Hour, Blue Hour, Rembrandt, Neon, Window Light, God Rays, Rim, Hard |

### 🎭 표현 & 연출 (섹션 9~12)

| 섹션 | 내용 | 주요 용어 |
|------|------|-----------|
| 표정 / 감정 | 얼굴 표정 (SVG 얼굴 일러스트 포함) | 미소, 환한 웃음, 노려보기, 눈물, 공포, 사색, 수수께끼, 도전적 |
| 시선 · 자세 · 동작 | 눈/몸/움직임 | 정면 응시, 곁눈질, 팔짱, 콘트라포스토, 히어로 워크, 슬로모션 |
| 분위기 · 연출 | 무드 + 영화 기법 + 감독 레퍼 | 멜랑콜리, 에픽, 느와르, 미장센, 이중노출, 웨스 앤더슨, 큐브릭 |
| 질감 · 환경 · 시대 | 표면/날씨/세계관 | 젖은 피부, 폭우, 안개, 벚꽃, 빅토리안, 사이버펑크, 지브리 |

## 사용법

1. `AI_Visual_Prompt_Master_Guide.html`을 다운로드한다
2. 브라우저에서 연다 (Chrome, Safari, Edge 등)
3. 원하는 분위기/구도에 해당하는 용어를 찾는다
4. 각 카드의 `PROMPT` 부분을 복사하거나 참고하여 프롬프트를 조합한다

### 프롬프트 조합 공식

용어를 레이어처럼 쌓으면 된다:

```
[샷 사이즈] + [앵글] + [렌즈] + [필름/색보정] + [조명] + [표정] + [동작] + [분위기] + [피사체 묘사]
```

### 조합 예시

**시네마틱 인물 — 할리우드 톤**
```
medium close-up, low angle, shallow depth of field, anamorphic lens flare,
teal and orange color grading, Rembrandt lighting,
a confident businesswoman walking through a modern lobby
```

**야간 사이버펑크**
```
wide shot, dutch angle, shot on CineStill 800T, red halation,
neon cyberpunk lighting, rain-soaked street,
tracking shot through a neon-lit Tokyo alley at night
```

**빈티지 카페 인물**
```
close-up, Fujifilm Nostalgic Negative, warm amber highlights,
natural window light, pensive expression, looking away,
a woman reading a letter in a sunlit vintage cafe
```

## 누구를 위한 건가

- "cinematic"만 반복하다가 한계를 느낀 사람
- 프롬프트 어휘력을 한 단계 올리고 싶은 사람
- 영화/사진 용어를 체계적으로 참조하고 싶은 사람
